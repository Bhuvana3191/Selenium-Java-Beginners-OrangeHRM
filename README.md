## Project Description

This project is a simple Selenium automation framework developed using **Java** to automate the **OrangeHRM** web application.

The purpose of this project is to practice and demonstrate:

- Selenium WebDriver
- Test automation using Java
- TestNG framework
- Page Object Model (POM)

---

## Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven
- Eclipse IDE
- Git & GitHub

---

## Automated Test Scenarios

- Login with valid credentials
- Login with invalid credentials
- Basic page navigation validation

---

## How to Run the Project

1. Clone the repository:
   git clone https://github.com/Bhuvana3191/Selenium-Java-OrangeHRM.git
2. Open the project in Eclipse
3. Ensure Java and Maven are installed
4. Run the tests by:

- Running individual TestNG test classes
- (Optional) Using a TestNG XML suite file if configured
